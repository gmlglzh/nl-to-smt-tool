📚 Template Reference

Available constraint templates:

1. always_previously_requires

Semantics: Event A must occur before event B

trigger requires all_before + one_of_before groups

2. state_persistence_until

Semantics: State persists from start_event until end_event

start_event → state persists → end_event

3. always_prevents_while_state

Semantics: State blocks event from occurring

state → NOT(event)

4. always_within_requires

Semantics: Trigger requires required event within N steps

trigger → required within time window

5. always_next_requires

Semantics: Event triggers state in next step

event[t] → state[t+1]

6. event_requires_state

Semantics: Event can only occur when state is true/false

event → state == value

7. state_implication

Semantics: One state implies another state

if_state == if_value → then_state == then_value

8. state_blocks_event

Semantics: State blocks event

state == value → NOT(event)
